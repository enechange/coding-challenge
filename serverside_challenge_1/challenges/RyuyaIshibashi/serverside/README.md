## Usage
* URL  
`https://ryuya-serverside.herokuapp.com/api/calculation?ampare=#{契約アンペア数}&amount=#{使用料}`

* 上記URLへのGETリクエストで動作をご確認いただけます。 

* `#{契約アンペア数}`、`#{使用料}`の箇所にそれぞれ適当な整数値を設定ください。  

* その他詳細については、設計書をご覧ください。
